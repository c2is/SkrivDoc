SkrivDoc

Build technical documentations easily using skriv markup language.

What is Skriv

Skriv is a markup language usefull to write some texts, especially documentations, with layout informations which can be rendered in many formats (html, pdf...).

What is SkrivDoc

SkrivDoc is an environment ready to use which allow you to :

  • Create, delete and edit skriv pages,
  • Render them in html on the fly,
  • Export all html contents as a simple website with table of content and publish it on github pages.

How to use SkrivDoc

Clone, set yours informations and instal

  1. git clone git@github.com:c2is/SkrivDoc.git YourDocumentation
  2. cd YourDocumentation
  3. git remote set-url origin git@github.com:account/YourDocumentation.git
  4. git submodule init
  5. git submodule update

Launch the editor

  1. ./edit
